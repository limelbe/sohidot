Animation


animation

animation : {name} {duration} {timing-function} {delay} {iteration-count} {direction} {fill-mode} {play-state}
Property Description
animation-name @keyframes에 지정된 이름을 설정합니다.
animation-duration 애니메이션이 실행되는 시간을 설정합니다.
animation-timing-function 애니메이션 키프레임 움직임을 설정합니다.
animation-delay 애니메이션이 로드된 후 애니메이션이 다시 시작될 때까지의 시간을 설정합니다.
animation-iteration-count 애니메이션이 반복되는 횟수를 설정합니다.
animation-direction 애니메이션의 움직임 방향을 설정합니다.
animation-fill-mode 애니메이션이 끝난 후에 상태를 설정합니다.
animation-play-state 애니메이션의 진행 상태를 설정합니다.

timing-function

Property Description
liner 일정한 간격으로 움직입니다.
ease 처음에 가속하고 마지막 부분에서 급격히 감소합니다.
ease-in 처음에 천천히 시작하고 마지막 부분에서 가속합니다.
ease-out 처음에 최대 속도로 시작하고 마지막에 감속합니다.
ease-in-out 처음에는 제로 속도에서 시작하고 중간 지점에서 최고 속도로 움직이다가 마지막 부분에서 서서히 감속합니다.
step-start 애니메이션이 시작점에서만 표현됩니다.
step-end 애니메이션이 끝점에서만 표현됩니다.
steps(ints, start/end) 애니메이션을 단계별로 표현합니다.
cubic-bezier(n,n,n,n) 베지어 곡선 값을 만들어서 속도를 설정합니다.

transition

transiton : {property} {duration} {timing-function} {dealy}
Property Description
transiton-property 트랜지션을 적용할 css 속성 대상을 설정합니다.
transiton-duration 트랜지션의 작동 시간을 설정합니다.
transiton-timing-function 트랜지션의 움직임 효과를 설정합니다.
transtion-delay 트랜지션이 시작되기 전 대기시간을 설정합니다.
